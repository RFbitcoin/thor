THOR — Bitcoin Trading Dashboard

THOR is a professional Bitcoin and cryptocurrency trading dashboard built by RFBitcoin. JavaScript is required to use the live dashboard.

What THOR Does

  • 35-point Bitcoin signal scoring system combining technical, macro, and on-chain data
  • Real-time BTC and ETH price feeds via Kraken and Binance
  • Macro economic indicators: VIX, DXY Dollar Index, M2 money supply
  • Bitcoin dominance and market cap tracking
  • On-chain network health: hash rate, mempool, fee pressure
  • Volatility regime detection using ATR percentile analysis
  • Paper trading simulator for strategy testing
  • Live Kraken exchange integration for real trades
  • AI-powered price prediction and trend analysis
  • Backtesting engine for historical signal validation
  • Dollar-cost averaging (DCA) strategy tracker
